# AurionOS Installer

> Planned for v0.5+. Not part of MVP.

## Strategy

### MVP
No custom installer. Use the live ISO with manual installation or Calamares.

### Production
Calamares-based installer with Aurion branding:
- Custom QML slides during installation
- Btrfs as default filesystem
- Partition layout optimized for snapshots
- AI hardware check during installation
- Post-install setup wizard
